1. Click a GridFS bucket name to view its files in the Query Response pane.

  1. You can perform the following actions:

 

Functionality Description
Define Query Add selectors/conditions in the query box to filter results.
Skip Skip a specified number of files from start.
Max page size View more or less files per page.
Sort by fields Sort results on the fields specified.
Open a file Click the file name in the Tree Table view. Alternatively, click Open in JSON view.
Download a file Click icon in Tree Table view. Alternatively, click Download in JSON view.
Delete a file Click icon in Tree Table view. Alternatively, click Delete in JSON view.
