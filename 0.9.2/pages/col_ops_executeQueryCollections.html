mViewer has the ability to execute queries from its Query Executor, thus eliminating the dependency on MongoDB terminal to execute queries.

 

Query Executor is displayed for a particular collection but can perform operations on other databases or collections as well.

 

  1. Simply click on a collection name to access Query Executor.
  2. By default, sub query for that collection along with all the documents of that collection is displayed in the Query Response pane.
  3. You can also select attributes, skip records, change page size and sort by fields.
  4. Results can be viewed in JSON or Tree Table format.
