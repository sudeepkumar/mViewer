1. Click a collection name to view its documents in the Query Response pane.

 

Results can be filtered using the following fields:

 

Functionality Description
Define Query Add selectors/conditions in the query box to filter results.
Skip

Skip a specified number of documents from start.
Max page size View more or less documents per page.
Sort by fields Sort results on the fields specified.
Attributes Select or deselect specific documents.

 

 

  1. Click Execute Query

 

This updates results on modifying query, skip, max page size, sort by fields.

 

  1. Click First, Previous, Next and Last to navigate to a particular set of documents in the collection.
