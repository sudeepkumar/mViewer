Spotlight Search > For Multiple Elements
  1. In the Spotlight Search command bar, type 3 letters to search. For instance, type "new".

 

Multiple Elements are highlighted in blue. In this case, NewDataBase and NewCollection are highlighted.

By default, the first element on the page gets selected. (Element name with white text is the element highlighted)

 

Use Up, Down, Right and Left arrow keys to navigate to the desired element.
In this case, NewCollection is selected by clicking on down or Right arrow keys and the selected element text is highlighted in white.

  1. Press the key.

 

The collection page is displayed.

 

Similarly, you can open any visible database, collection, grid bucket, user and index, link, button or text field present on the page.
