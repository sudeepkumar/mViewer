1. Hover on a database name. Click on the menu.
  2. Click Add Collection.

  1. Enter the new collection name and click Submit.

 

  1. Enter the following details:

 

Field Description
Name Enter the new name of the collection
Capped

Check this box to create a fixed size collection
Size Enter the size in bytes
Max Documents Type the maximum number of documents to be loaded

 

 

  1. Click Submit

 

The collection is now created.
