mViewer has the ability to execute queries from its Query Executor, thus eliminating the dependency on MongoDB terminal to execute queries.

 

Query Executor is displayed for a particular database but can perform operations on other databases or collections as well.

 

  1. Simply click on a database name to access Query Executor.
  2. By default, query for database statistics is displayed along with results in the Query Response pane.
  3. You can also skip records, change page size and sort by fields.
  4. Results can be viewed in JSON or Tree Table format.
